
# 3D Interactive Software Engineering Portfolio 🚀

Welcome to my interactive 3D portfolio website! Built using modern web technologies and 3D rendering frameworks, this project serves as a dynamic workspace to showcase my software engineering journey, core development skill sets, certifications, and academic/entrepreneurial projects.

---

## 🛠️ Tech Stack & Architecture

This portfolio brings together smooth interactive animations and optimized frontend performance:
* **Frontend Framework:** React.js / Next.js
* **3D Rendering Engine:** Three.js / @react-three/fiber & @react-three/drei
* **Styling & Layout:** Tailwind CSS
* **Build & Deployment:** Node.js environment managed within the Antigravity IDE
